# Solar System 3D

Інтерактивна 3D-візуалізація Сонячної системи у браузері, реалізована на [Three.js](https://threejs.org/). Сонце та вісім планет обертаються навколо центру по власних орбітах, а при виборі планети камера переходить у режим обльоту навколо неї з відображенням довідкової інформації.

## Можливості

- Сонце з пульсуючим світінням (glow) та анімованими частинками атмосфери.
- Восьмипланетна система (Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, Neptune) з реалістичними текстурами та індивідуальною швидкістю обертання й орбітальної ротації.
- Орбітальні кільця навколо Сонця для кожної планети.
- Анімоване поле зірок на фоні.
- Панель керування з кнопками для кожної планети та Сонця.
- Інформаційна панель з описом, типом, діаметром, масою, відстанню до Сонця, орбітальним періодом і температурою обраної планети.
- Камера, що плавно облітає вибрану планету, та масштабування сцени колесом миші.

## Технології

- [Three.js](https://threejs.org/) — рендеринг 3D-сцени (WebGL).
- [jQuery](https://jquery.com/) — робота з DOM та UI-елементами.
- [Vite](https://vitejs.dev/) — дев-сервер та збірка проєкту.

## Технічні рішення

- **Рендеринг та камера** — `THREE.WebGLRenderer` з `antialias: true` і `setPixelRatio`, `THREE.PerspectiveCamera` (fov 75) для перспективної проєкції сцени (`src/settings.js`).
- **Геометрія планет і Сонця** — одна спільна `THREE.SphereGeometry` (`src/baseObjects.js`), яка переюзається для всіх тіл через `THREE.MeshBasicMaterial` з накладеною текстурою (`THREE.TextureLoader`), що економить пам'ять і код.
- **Ієрархія об'єктів для орбіт** — кожна планета додається не напряму в сцену, а в обгортку `THREE.Object3D` (`planetGroup`); обертання групи (`planetGroup.rotation.y`) дає орбітальний рух навколо Сонця, а власне обертання `planet.rotation.y` — обертання навколо власної осі (`src/planets.js`).
- **Орбітальні кільця** — `THREE.RingGeometry` з тонким радіусом (`radius ± 0.1`) і прозорим `MeshBasicMaterial` (`side: THREE.DoubleSide`) для відображення траєкторій планет.
- **Світіння Сонця** — `THREE.Sprite` з `SpriteMaterial` і `blending: THREE.AdditiveBlending`, масштаб і прозорість якого пульсують у часі через `Math.sin`, створюючи ефект "дихання" (`src/sun.js`, `src/main.js`).
- **Зорове поле та частинки атмосфери Сонця** — процедурно згенеровані хмари точок на `THREE.BufferGeometry` + `THREE.Float32BufferAttribute` (позиції рандомізовані `Math.random()`), що рендеряться через `THREE.Points`/`PointsMaterial`. Кожен кадр координати точок оновлюються вручну (`needsUpdate = true`) для ефекту руху зірок і розльоту частинок.
- **Цикл анімації** — `renderer.setAnimationLoop`, який щокадру оновлює позиції планет, зірок, частинок і камери — без використання `requestAnimationFrame` напряму.
- **Камера слідкування за планетою** — при виборі планети камера обчислює світову позицію об'єкта (`getWorldPosition`) і рухається по колу навколо неї (`Math.cos`/`Math.sin` від часу), викликаючи `camera.lookAt`.
- **Керування зумом** — слухач події `wheel` змінює цільову Z-позицію камери, яка потім плавно інтерполюється (`lerp`-подібна формула `cameraZ += (target - cameraZ) * 0.05`) для плавного наближення/віддалення.
- **UI поверх Canvas** — DOM-елементи (кнопки планет, інформаційна панель) створюються та оновлюються через jQuery (`$("<button>")`, `.text()`, `.fadeIn()/.fadeOut()`), а не засобами Three.js, тобто інтерфейс і 3D-сцена розділені.

## Структура проєкту

```
index.html              # точка входу, розмітка інформаційної панелі
src/
  main.js               # ініціалізація сцени, UI, основний цикл анімації
  settings.js           # рендерер, сцена, камера, завантажувач текстур
  baseObjects.js        # базова геометрія сфери та контейнер solarSystem
  sun.js                 # Сонце, його світіння та частинки
  stars.js               # фонове поле зірок
  planets.js             # створення планет, орбіт та довідкова інформація
  style/style.css         # стилі інтерфейсу
  textures/              # текстури планет і Сонця
```

## Вимоги

- [Node.js](https://nodejs.org/) (рекомендовано версія 18+)
- npm

## Встановлення

```bash
npm install
```

## Запуск

```bash
npm start
```

Команда запускає локальний дев-сервер Vite (`npx vite`). Після старту перейдіть за адресою, яку Vite виведе в консолі (зазвичай `http://localhost:5173`).

## Керування

- **Кнопки планет/Сонця** — переміщують камеру до обраного об'єкта та показують інформаційну панель (для Сонця панель приховується).
- **Колесо миші** — наближення/віддалення камери в режимі огляду всієї системи.
